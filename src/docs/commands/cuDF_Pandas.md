## Run GPU Accelerated Pandas using cuDF

For IPython files/Jupyter Notebooks (.ipynb file)
```python
%load_ext cudf.pandas
import pandas as pd
```

<br /> 

For Python script (.py file)

```python
python3 -m cudf.pandas <name_of_python_script>.py
```
