_This is for **Windows**_

Make sure Git Bash is installed

1. Open VS Code
2. Press `Ctrl + Shift + P` or `F1`
3. Type `Terminal: Select Default Profile`
4. Select Git Bash
5. Open a new terminal with **Ctrl + `** (backtick)
